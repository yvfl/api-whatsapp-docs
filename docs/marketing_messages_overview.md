# API de Mensagens de Marketing Lite

Updated: 30 de out de 2025

Válido a partir de 1º de julho de 2025: os preços por mensagem já estão em vigor. Além disso, as taxas para mensagens de marketing na API de Nuvem e na API de MM Lite são consistentes, de acordo com as tabelas de tarifas publicadas [neste link](/documentation/business-messaging/whatsapp/pricing).

Planejamos lançar a MM Lite para o público geral no quarto trimestre de 2025.

A API de Mensagens de Marketing Lite ("API de MM Lite") é uma solução fácil de usar, desenvolvida para profissionais de marketing direto ou CRM, com o objetivo de aprimorar a experiência do cliente e gerar resultados iguais ou superiores para empresas quando comparada com o marketing feito usando a API de Nuvem.

### Principais benefícios

-   **A API de MM Lite entrega um número igual ou maior de mensagens de marketing do que a API de Nuvem.** A API de MM Lite oferece limites de mensagens mais dinâmicos para que as mensagens de marketing com alto engajamento (por exemplo, mensagens mais lidas) alcancem um número maior de clientes. Na Índia, as mensagens de marketing enviadas pela API de MM Lite que apresentaram maior engajamento (por exemplo, leituras) registraram até 9% mais entregas em comparação com a API de Nuvem ([1](#footnotes)).-   **A API de MM Lite tem novos [recursos](/documentation/business-messaging/whatsapp/marketing-messages/features) exclusivos que não estão disponíveis na API de Nuvem.** A API de MM Lite oferece referências de comparação exclusivas que mostram o desempenho dos seus modelos em relação a modelos semelhantes na sua região, além de recomendações sobre como melhorar as métricas de desempenho, mensagens de marketing com validade programada para promoções de curta duração, novos relatórios de conversão para mostrar conversões na web e no app e muito mais. Confira a [lista completa de recursos da API de MM Lite](/documentation/business-messaging/whatsapp/marketing-messages/features).-   **É fácil fazer a transição da API de Nuvem.** As empresas podem usar contas do WhatsApp Business, números de telefone e modelos de marketing que já existem. Os parceiros continuam com o mesmo modelo de cobrança da API de Nuvem e têm uma integração simples e sem complicações.

## Notas de rodapé

-   \*Teste A/B realizado com aproximadamente 12 milhões de mensagens de marketing entregues, enviadas por anunciantes na Índia, entre 1º e 31 de janeiro de 2025. O teste comparou a entrega otimizada da API de MM Lite com a entrega padrão da API de Nuvem apenas entre mensagens de alto engajamento (por exemplo, mensagens com mais leituras, cliques e assim por diante). A análise consistiu em um teste t com 95% de confiança.
