# API de Mensagens de Marketing para o WhatsApp

Updated: 10 de fev de 2026

A API de Mensagens de Marketing para o WhatsApp (anteriormente conhecida como API de Mensagens de Marketing Lite) agora está disponível para todos.

A API de MM para o WhatsApp é a nossa solução de marketing de última geração criada para melhorar a experiência do cliente e entregar a mensagem certa para mais pessoas.

### Principais benefícios

-   **Impulsione e mensure os resultados comerciais**: com nossas otimizações automáticas de entrega, alcance mais pessoas que considerarão suas mensagens importantes, além de gerar mais leituras e cliques. Também é possível acessar insights de mensuração exclusivos:
    -   _Referências de desempenho_, para entender como foi o desempenho da sua mensagem em comparação com empresas semelhantes-   _Recomendações personalizadas_, para melhorar o desempenho da campanha-   **Aprimore a experiência e o engajamento dos clientes**: a API de MM para o WhatsApp ajuda a entregar mensagens de marketing mais relevantes e em tempo hábil aos clientes com recursos exclusivos, como:
    -   Otimizações automáticas de criativos (em testes) para aplicar tratamentos criativos, como animação de imagens e filtragem para mensagens mais envolventes.-   Formatos de mídia avançados, como GIFs.-   Tempo de vida, para evitar a entrega de mensagens irrelevantes ou atrasadas em campanhas urgentes.-   **Atualização fácil, com confiabilidade e segurança consistentes**: a API de MM para o WhatsApp é confiável, segura e fácil de usar. Essa opção oferece um esquema técnico semelhante e o mesmo modelo de cobrança da API de Nuvem. Além disso, as empresas podem usar números de telefone e modelos de mensagem de marketing existentes.

Envie todo o seu tráfego de marketing ao ponto de extremidade `/marketing_messages` para fazer o roteamento automático de mensagens de empresas qualificadas.

## Notas de rodapé

-   \*Teste A/B realizado com aproximadamente 12 milhões de mensagens de marketing entregues, enviadas por anunciantes na Índia, entre 1º e 31 de janeiro de 2025. O teste comparou a entrega otimizada da API de MM para o WhatsApp com a entrega padrão da API de Nuvem apenas entre mensagens de alto engajamento (por exemplo, mensagens com mais leituras, cliques e assim por diante). A análise consistiu em um teste t com 95% de confiança.
