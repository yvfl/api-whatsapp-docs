# Perguntas frequentes sobre a API de Grupos

Updated: 22 de out de 2025

### O que acontece quando excluo um grupo?

-   Nenhum membro, incluindo você, poderá enviar mensagens ao grupo.-   Caso a API de Nuvem tenha recebido mensagens ou status antes da exclusão do grupo, você ainda poderá receber webhooks relacionados a eles.

### Por que um participante não consegue entrar no grupo usando meu link de convite?

Estes são alguns dos possíveis motivos:

-   O link do convite pode ter sido excluído.-   Talvez você já tenha removido o participante do grupo.-   Talvez o grupo já esteja cheio.

### Como posso enviar meu link de convite aos usuários?

-   Você pode enviar o link de convite em uma conversa individual.-   Um novo modelo de utilidade está disponível na [Biblioteca de Modelos](https://business.facebook.com/wa/manage/template-library) para [enviar links de convite para o grupo](/documentation/business-messaging/whatsapp/groups/reference#send-group-invite-link-template-message).-   Você também pode criar modelos de marketing personalizados e de formato livre.

### Em quais países o recurso Grupos está disponível?

-   Os grupos estão disponíveis em [todos os países onde a API de Nuvem está disponível](/documentation/business-messaging/whatsapp/support#country-restrictions).
