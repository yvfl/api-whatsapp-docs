# Diretrizes de análise do app para a API de Ligações

Updated: 13 de nov de 2025

## Visão geral

Siga estas diretrizes para garantir a aprovação da sua inscrição na Análise do App ao usar os recursos da API de Ligações do WhatsApp Business.

## Orientações

### Para a permissão de gerenciamento do WhatsApp Business

Para mostrar claramente que o app pode habilitar e desabilitar recursos de ligação, informe se o ícone do botão de ligação está visível.

Compartilhe um vídeo que mostre você habilitando e desabilitando o ícone de botão de ligação para o WhatsApp Business por meio da solicitação CURL ou das configurações na interface do usuário do seu app.

Para isso, _ative e desative_ os recursos de ligação, em vez de apenas mudar a visibilidade do ícone do botão Ligar.

-   [Saiba como habilitar e desabilitar recursos da API de Ligações via API](/documentation/business-messaging/whatsapp/calling/call-settings#configure-update-business-phone-number-calling-settings)-   [Saiba como habilitar e desabilitar os recursos da API de Ligações no Gerenciador do WhatsApp](/documentation/business-messaging/whatsapp/calling/call-settings#call-settings-in-whatsapp-manager)

#### Exemplo

-   Exiba uma conversa entre a empresa e um usuário do WhatsApp que não tem o ícone do botão Ligar.-   Use o app do desenvolvedor para habilitar os recursos da API de Ligações no número de telefone comercial, o que habilitará o ícone do botão Ligar para o número especificado-   Volte para a mesma conversa e confirme que o ícone do botão Ligar está visível

### Para a permissão de gerenciamento de mensagens do WhatsApp

Demonstre claramente que seu app oferece suporte a **um dos seguintes casos de uso:**

#### Caso de uso 1: fazer uma ligação iniciada pela empresa

Compartilhe um vídeo que mostre você usando o app para fazer uma ligação iniciada pela empresa. Depois, mostre um usuário aceitando a ligação em um cliente móvel do WhatsApp.

#### Caso de uso 2: receber uma ligação iniciada pelo usuário

Compartilhe um vídeo de um usuário fazendo uma ligação para o número de telefone da sua empresa. Depois, mostre o app recebendo a ligação.

Exiba uma das seguintes opções:

-   A ligação recebida na interface do usuário do app WhatsApp-   O webhook da ligação conforme entregue ao seu app
