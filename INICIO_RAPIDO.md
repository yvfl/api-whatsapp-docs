## 🔄 Para Usar em Outro Projeto

### Opção Rápida: Link Simbólico

```bash
# No seu projeto
ln -s /caminho/para/docs-waba/docs ./docs-waba
```

Depois use: `@docs-waba/AGENTS.md`

### Opção Git: Submódulo

```bash
# No seu projeto
git submodule add <url-do-repo> docs-waba
```

Depois use: `@docs-waba/docs/AGENTS.md`
